# House_Price_Prediction

# 🏠 House Price Prediction Web App

A complete end-to-end Machine Learning web application that predicts house prices based on user inputs like area, number of bedrooms, and house age. This project integrates Machine Learning with a Flask web interface for real-time predictions.

---

## 🚀 Project Overview

This project demonstrates how to build, train, and deploy a Machine Learning model into a web application. Users can enter property details through a simple UI and get instant price predictions.

---

## 🎯 Key Features

* 📊 Real-time house price prediction
* 🌐 Interactive web interface using HTML
* ⚡ Fast predictions using trained ML model
* 🔗 Flask backend integration
* 📁 Clean project structure for scalability

---

## 🧠 Tech Stack

* **Programming Language:** Python
* **Machine Learning:** Scikit-learn
* **Backend Framework:** Flask
* **Frontend:** HTML, CSS
* **Libraries:** Pandas, NumPy, Pickle

## ⚙️ How It Works

1. User enters:

   * Area
   * Number of Bedrooms
   * Age of House

2. Data is sent to Flask backend

3. Model processes input

4. Prediction is returned and displayed on UI

## 🛠️ Installation & Setup

```bash

cd house-price-prediction
```

### 2️⃣ Install dependencies

```bash

```

```bash

```

```

```
